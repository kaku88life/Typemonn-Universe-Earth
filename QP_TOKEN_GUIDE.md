## 📌 What is QP?

**QP (Quantum Piece)** 是型月全息投影社群的虛擬獎勵代幣。

它代表貢獻者對社群的貢獻價值，用於：
- 🎮 遊戲化激勵社群參與
- 🏆 分級展示貢獻程度
- 🛍️ 兌換社群特權功能
- 📊 計算用戶信譽和排名

**重要**: QP **不是加密貨幣**，**不能交易、出售或兌換成現實貨幣**。

---

## 🎯 QP 的三大用途

### 1️⃣ **衡量貢獻** (Contribution Indicator)
- 反映用戶對社群的實際貢獻
- 越高的 QP 餘額 = 越活躍的貢獻者
- 在排行榜中計入總分

### 2️⃣ **社群治理** (Governance Power)
- 花費 QP 來加快提案投票速度
- 花費 QP 來锁定爭議內容
- 花費 QP 查看編輯者的詳細筆記

### 3️⃣ **社群榮譽** (Prestige Symbol)
- 高 QP 用戶獲得排行榜排名
- 解鎖「傳奇貢獻者」等徽章
- 被社群認可為知識權威

## 💰 QP 獲取方式

### A. 編輯貢獻

| 行為 | 獲得 QP | 說明 |
|------|---------|------|
| 首次編輯通過 | **10 QP** | 一次性獎勵 |
| 編輯通過 | **5 QP** | 每次編輯都有 |
| 大型編輯通過 | **25 QP** | 5+ 個變更 |
| 編輯被標記「有幫助」 | **3 QP** | 社群認可 |

**示例**:
```
新用戶 Amy 提交第一份編輯（位置描述）
✓ 編輯通過 → +10 QP (首次獎勵)
✓ 後來被 5 人標記「有幫助」 → +3×5 = +15 QP
總計: 25 QP
```

---

### B. 社群治理

| 行為 | 獲得 QP | 說明 |
|------|---------|------|
| 投票參與 | **0.5 QP** | 參與一次投票 |
| 投票正確 | **2 QP** | 投票與最終結果一致 |
| 發起爭議投票 | **0 QP** | 免費 |

**示例**:
```
編輯 Bob 對 50 個提案投票
✓ 每次投票 → 0.5 × 50 = 25 QP (基礎獎勵)
✓ 其中 30 票與結果一致 → 2 × 30 = 60 QP (獎勵)
總計: 85 QP
```

---

### C. 知識傳承

| 行為 | 獲得 QP | 說明 |
|------|---------|------|
| 編輯被引用 | **5 QP** | 其他人基於你的編輯繼續編輯 |
| 每日登錄 | **1 QP** | 連續登錄 |

**示例**:
```
Carol 的 Fuyuki 位置編輯被 8 人引用
✓ 5 × 8 = 40 QP (知識傳承獎勵)
```

---

### D. 日常獎勵

```
每日登錄: 1 QP
每日最高獲得: 50 QP
投票獎勵每日上限: 20 QP
```

## 💸 QP 花費方式

### 1. 加速提案投票

```
成本: 10 QP
效果: 投票期限 7 天 → 3 天
使用場景: 急需快速反饋的重要編輯
```

**何時使用？**
- 你的編輯很重要，需要快速批准
- 你想搶先佔得先機
- 社群有緊急情況

---

### 2. 官方推薦標籤

```
成本: 50 QP
效果: 為你的編輯添加「官方推薦」標籤
使用場景: 展示特別優質的編輯
```

**效果**:
- 在列表中突出顯示
- 提高被引用的可能性
- 增加社群信任度

---

### 3. 查看編輯筆記

```
成本: 5 QP
效果: 查看編輯者的詳細修改說明
使用場景: 了解為什麼要這樣編輯
```

**內容包括**:
- 編輯理由
- 數據來源
- 參考資料鏈接
- 編輯者的個人筆記

---

### 4. 鎖定提案

```
成本: 20 QP
效果: 防止該提案被進一步編輯（1 週）
使用場景: 解決完全爭議，防止重複修改
```

**使用條件**:
- 提案已通過
- 你是編輯者或社群達成共識
- 可被仲裁員解除

---

### 5. 發起爭議投票

```
成本: 0 QP (免費)
效果: 質疑某個編輯的準確性
使用場景: 懷疑內容不准確
```

**流程**:
1. 免費發起爭議投票
2. 社群投票決定該編輯是否有效
3. 如果無效，編輯被回退，編輯者失去信譽

## ⚙️ QP 經濟設計原則

### 1. **平衡性** (Balance)
- 獲得和花費的 QP 數量相當
- 避免某些用戶積累過多 QP
- 定期調整獎勵金額

### 2. **可獲得性** (Accessibility)
- 新手也能快速獲得 QP
- 日常活動都能賺取 QP
- 不依賴金錢投資

### 3. **有意義性** (Meaningfulness)
- QP 花費必須有實際意義
- 不能「無用功」地浪費
- 社群能看到 QP 的效果

### 4. **可持續性** (Sustainability)
- QP 不會無限膨脹
- 定期銷毀機制（1% 的花費）
- 月度獲得上限

---

## 🛡️ 防止 QP 濫用

### 禁止行為

❌ **不允許**:
- QP 交易（用戶間轉移）
- QP 兌換現金
- 機械地重複操作來積累 QP
- 互相投票 (刷票)

✅ **允許**:
- 正常的編輯和投票
- 合法的 QP 花費
- 社群治理參與

### 違規後果

```
輕度違規 (自動化刷票):
→ 警告 + 扣除 QP + 臨時禁言

中度違規 (小規模販賣 QP):
→ 扣除所有 QP + 1 週禁止投票

重度違規 (組織大規模詐欺):
→ 永久封禁 + 負聲譽 -500
```

## 💬 常見問題

### Q: 為什麼叫 QP?
**A**: QP 代表「Quantum Piece」（量子碎片）。
- 象徵小而精的貢獻
- 與型月科幻概念相符
- 簡短易記

### Q: QP 會貶值嗎?
**A**: 不會。
- 總供應量有上限（100 萬 QP）
- 1% 的花費會被銷毀，實際上在減少
- QP 作為信譽證明，價值穩定

### Q: 能把 QP 給別人嗎?
**A**: 不能。
- 這是個人貢獻的證明
- 無法轉移防止詐欺
- 鼓勵誠實參與

### Q: 刪除賬戶時 QP 怎麼辦?
**A**: QP 會被銷毀。
- 退出社群的用戶失去 QP
- 被銷毀的 QP 減少總供應量
- 保護社群的 QP 價值

### Q: QP 對我有什麼好處？
**A**: 多方面好處！
- 📊 排行榜排名（社群認可）
- 🎮 解鎖成就和徽章
- 🛍️ 兌換特權功能
- 💬 提高社群發言權
- 🏆 成為「傳奇貢獻者」
